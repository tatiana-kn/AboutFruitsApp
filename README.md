## AboutFruitsApp

### Project Description
iOS app that provides information about various fruits and their nutrition using data from the Fruityvice REST API.

### Built using:
- Swift
- UIKit
- MVC (with NetworkClient and Configurator)
- AutoLayout
- URLSession
